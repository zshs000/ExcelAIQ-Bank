## 3. 改动明细

## 3.1 QuestionService.java

- 增加 `java.util.Map` 导入，修复接口中 `Map<String, String>` 无法解析导致的编译错误。
- 移除未使用的 `org.springframework.stereotype.Service` 导入。

## 3.2 QuestionServiceImpl.java

- 修复分页查询对象拷贝方向错误：
  - `BeanUtils.copyProperties(questionDO, request)` -> `BeanUtils.copyProperties(request, questionDO)`
- 修复空指针风险：
  - `sendQuestionsToQueue` 中先进行 `questionIds` 判空，再打印 `questionIds.size()`。
- 修复 AI 回写更新不落库问题：
  - `updateQuestionStatusToReview` 从 `updateByPrimaryKey` 改为 `updateByPrimaryKeySelective`，保证 `answer` 可更新。
- 移除对不存在能力的调用（编译/运行风险点）：
  - 删除 `questionDOMapper.updateBatch(...)` 调用，改为逐条 `updateByPrimaryKeySelective(...)`。
  - 删除 `updateDO.setErrorMessage(...)`（`QuestionDO` 中无该字段）。
- `batchUpdateSuccessQuestions` 与 `batchUpdateFailedQuestions` 改为逐条更新并统计成功数，避免依赖未定义 Mapper 方法。
